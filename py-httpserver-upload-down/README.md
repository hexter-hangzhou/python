一个简单的python代码，实现简单的http服务，在服务器运行后可以上传下载文件到服务器。

比如手机上的文件照片传的服务器，就可以手机浏览器这个网页来上传

使用方法：python httpsv.py 端口号

如：python httpsv.py 7600
